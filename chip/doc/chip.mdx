# Chip

Chips can be used to display entities in small blocks.

## Usage

```tsx
import * as React from 'react';
import { Chip } from '@divriots/starter-origami/chip';

const MyComponent = () => (
  <Chip icon="information" onPress={() => console.log('Pressed')}>
    Example Chip
  </Chip>
);
```
