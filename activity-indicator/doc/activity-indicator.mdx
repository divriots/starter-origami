# Activity Indicator

Activity indicator is used to present progress of some activity in the app. It can be used as a drop-in for the ActivityIndicator shipped with React Native.

## Usage

```tsx
import * as React from 'react';
import { ActivityIndicator } from '@divriots/starter-origami/activity-indicator';

const MyComponent = () => (
  <ActivityIndicator animating={true} color="#c62828" />
);
```
