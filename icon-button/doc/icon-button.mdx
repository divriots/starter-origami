# IconButton

An icon button is a button which displays only an icon without a label. By default button has 150% size of the icon.

## Usage

```tsx
import * as React from 'react';
import { IconButton } from '@divriots/starter-origami/icon-button';

const MyComponent = () => (
  <IconButton icon="camera" size={20} onPress={() => console.log('Pressed')} />
);
```
